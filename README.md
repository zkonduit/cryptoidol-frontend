# cryptoidol-frontend

## Overview
Sing to a voice rating bot and publish your ratings onchain.
The site is hosted on [https://cryptoidol.tech](https://cryptoidol.tech)

## Quickstart for Development

Note we use `node >= 20`

```
yarn

yarn dev
```
